Weather API

Retrieve the data for a station for a specific date

URL format: http://127.0.0.1:8080/api/v1/station/date

URL Example: http://127.0.0.1:8080/api/v1/10/1988-10-25

Retrieve all the data for a station for all dates

URL format: http://127.0.0.1:8080/api/v2/station

URL Example: http://127.0.0.1:8080/api/v2/10

Retrieve all the data for a station for a specific year

URL format: http://127.0.0.1:8080/api/v3/year

URL Example: http://127.0.0.1:8080/api/v3/10/1990
